## Application 类

路径：Illuminate/Foundation/Application.php

```php
public function __construct($basePath = null)
{
    $this->registerBaseBindings();

    $this->registerBaseServiceProviders();

    $this->registerCoreContainerAliases();

    if ($basePath) {
        $this->setBasePath($basePath);
    }
}

protected function registerBaseBindings()
{
    static::setInstance($this);

    $this->instance('app', $this);

    $this->instance('Illuminate\Container\Container', $this);
}

protected function registerBaseServiceProviders()
{
    $this->register(new EventServiceProvider($this));

    $this->register(new RoutingServiceProvider($this));
}
```

1. registerBaseBindings 设置了 Application 的单例存储变量 instance；将 $this 绑定到 instance['app] 变量；将 $this 绑定到 instance['Illuminate\Container\Container'] 变量
2. registerBaseServiceProviders 注册两个基本的服务提供者 EventServiceProvider、RoutingServiceProvider
3. registerCoreContainerAliases 注册核心容器的别名
